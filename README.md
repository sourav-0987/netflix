## 📖 Project Overview

This project analyzes Netflix Movies and TV Shows data using Python to uncover content trends, genre popularity, release patterns, and country-wise distribution. The objective is to transform raw Netflix data into meaningful insights through data cleaning, exploratory data analysis (EDA), and visualization.

The project demonstrates practical data analysis skills using Python libraries such as Pandas, NumPy, Matplotlib, and Seaborn.

---

## 🎯 Business Objectives

* Analyze Netflix content distribution.
* Compare Movies vs TV Shows.
* Identify top content-producing countries.
* Discover popular genres and categories.
* Analyze release year trends.
* Understand content ratings distribution.

## 📈 Key Insights

* Movies represent the majority of Netflix content compared to TV Shows.
* Content production increased significantly after 2015.
* The United States contributes the highest number of titles on Netflix.
* International Movies and Dramas are among the most popular genres.
* TV-MA is the most common content rating across the platform.
